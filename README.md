# 📊 Dashboard Financeiro Pessoal

Dashboard financeiro completo com integração ao Google Sheets para controle de finanças pessoais.

## 🌟 Funcionalidades

- 📥 Registro de entradas e saídas
- 💳 Controle de múltiplos cartões
- 📊 Gráficos interativos mensais
- 📈 Análises BI completas
- 🔗 Integração com Google Sheets
- 💾 Armazenamento local e na nuvem
- 📱 Design responsivo e moderno

## 🚀 Tecnologias

- HTML5
- CSS3 (Design Moderno)
- JavaScript (Vanilla)
- Chart.js (Gráficos)
- Google Sheets API
- LocalStorage
